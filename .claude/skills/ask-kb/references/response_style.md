# 知识库问答回答风格指南

## 格式规则

| 场景 | 格式 |
|------|------|
| 核心答案 | 开头一句话直接给结论，再展开 |
| 多方案对比 | 使用对比表格 |
| 数学公式 | LaTeX 行内 `$...$` 或块级 `$$...$$` |
| 代码示例 | 代码块 + 语言标注 |
| 来源标注 | 明确注明"知识库"或"搜索结果" |

## 来源引用格式

**知识库命中时：**
```
根据知识库（后训练 > RLHF > PPO）：
[内容]
```

**知识库未命中时：**
```
知识库中暂无此内容的记录。

基于通用知识：
[内容]

需要联网搜索最新资料吗？
```

**混合来源时：**
```
**知识库内容：**
[来自知识库的部分]

**搜索补充（来源：URL）：**
[来自联网的部分]
```

## 问答结束后的归档提示

在一轮深度讨论结束时，如果产生了新的知识要点，可以简短提示：

> "本次讨论产生了一些新内容，如需保存到知识库，直接说'归档'即可。"

**不要主动执行归档**，等用户明确说"归档"再触发 archive skill。
